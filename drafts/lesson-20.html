Задание 6 (Анализ программного кода)

Условный оператор (if ... else ...) Повторение

"if" в переводе с анлийского -- "если", "else" -- "иначе"

Если выполняется условие, произвести перечисленные действия

Например, для такого кода:

x = -5
if x < 0:
    print("x - отрицательное число")
else:
    print("x - неотрицательное число")

Результат работы будет такой:

x - отрицательное число

Три разновидности программного кода в задачах ОГЭ

Условный оператор (if ... else ...)
Конечный цикл (for ...)
Бесконечный цикл (while ...)
