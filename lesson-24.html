# все комбинации: 100 101 102 .... 998 999
# t - время на одну комбинацию
# вывести список чисел с неповторяющимися цифрами
# вывести суммарное время z
t = 11
n = 0
z = 0
for k in range(100, 1000):
    a, b, c = str(k)
    if a != b and b != c and c != a :
        print(k, end=" ")
        z += t
        n += 1
h = z // 3600
m = z % 3600 // 60
s = z % 60
print()

print(f"На ввод всех {n} чисел: {z} секунд ({h}:{m}:{s})")

Задача 5:

Решение:

# Степени двойки: 1 2 4 8 16 32 64 ...
# Степени тройки: 1 3 9 27 81 243 729...
# число V вводится с клавиатуры
v = abs(float(input("Введите число:")))

n = 0  # переменная для степени двойки
while v > 2**n:
    n += 1
print(f"Число {v} находится между 2^{n-1} и 2^{n}. Едем в Москву!")

m = 0  # переменная для степени тройки
while v > 3**m:
    m += 1
print(f"Число {v} находится между 3^{m-1} и 3^{m}. Едем в Питер!")

Введите число:55
Число 55.0 находится между 2^5 и 2^6. Едем в Москву!
Число 55.0 находится между 3^3 и 3^4. Едем в Питер!
